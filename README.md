# Hicoder&IE 合同ハッカソン チーム2

### フロントエンド
1. 移動
```
cd frontend
```
2. パッケージのインストール
```
npm install
```
3. 起動
```
npm start
```

### バックエンド
1. 移動
```
cd backend
```
2. 仮想環境を有効にする
```
python -m venv .venv
source .venv/bin/activate  # (Windowsの場合は .venv\Scripts\activate)
```
3. パッケージのインストール
```
pip install -r requirements.txt
```
4. 起動
```
python .\main.py
```
